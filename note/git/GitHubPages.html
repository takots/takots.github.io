创建新的仓库
New Repositories
访问 GitHub。
在页面右上角，点击 "New" 按钮创建一个新的仓库。
Repository name 使用格式 username.github.io（例如，takots.github.io）。
添加页面内容
Create New File
在新仓库中，点击 "Add file"，选择 "Create new file"。
文件名命名为 index.html。
编写你的页面内容，并点击 "Commit new file" 保存。
启用 GitHub Pages
Enable GitHub Pages
进入你的仓库，点击 "Settings" 选项卡。
在左侧菜单中选择 "Pages"。
在 "Source" 部分选择 "master" 分支，并保存。
访问你的 GitPages
https://username.github.io/
你的 GitPage 将会在格式为 https://username.github.io/ 的地址上可用。
如果你的仓库名是 a01.github.io，
则 GitPage 地址为 https://username.github.io/a01.github.io/。
切换到 master 分支
Checkout To Other Branch
如果没有提交过任何内容，master 分支可能会被 GitHub 默认命名为 main。
在仓库页面中，点击 "Code" 按钮，选择 "main" 分支，并选择 "master" 分支。
切换到 master 分支后，你可以在 main 分支中选择 master 分支。
